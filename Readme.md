# Tic Tac toe Game

## Description
Tic tac toe game using web technologies

## Features
- Tic tac toe Game web application
- Play vs Computer
- Game History
- Load and Delete previous games

## Intallation
```shell script
pip install -r requirements.txt
```

## Run Project
```shell script
python uvicorn backend.main:app
```

## Test Project
```shell script
python pytest backend/test_main.py
```
